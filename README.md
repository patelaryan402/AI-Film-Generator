# 🎬 AI Short Film Generator

An AI-powered web application that generates short film content (story, script, scenes, narration) from a user’s idea.

## 🚀 Features

* Story Development
* Script Generation
* Scene Breakdown
* Narration Creation
* Typewriter Animation UI

## 🛠️ Tech Stack

* Java (Backend)
* HTML, CSS, JavaScript (Frontend)
* Gemini API (Google AI)

## 🔐 Security

API keys are securely managed using environment variables.

## ▶️ How to Run

1. Compile:
   javac -cp ".;lib/*" -d out src*.java

2. Run:
   java -cp "out;lib/*" Main

3. Open:
   http://localhost:8080
